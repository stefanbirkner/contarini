Contarini – Details

Information about usage of Contarini's classes and tags.

WebCrawlerInfo

A WebCrawlerInfo object stores all the meta information provided by a single page. It is a value object with a fluent interface. Create an empty object with new WebCrawlerInfo() and get richer info objects by calling its with-methods.

WebCrawlerInfo info = new WebCrawlerInfo().withCanonical("myCanonical.html");

WebCrawlerInfoRenderer

The WebCrawlerInfoRenderer render the meta tags with the information of a WebCrawlerInfo to an arbitrary Writer. 

new WebCrawlerInfoRenderer().writeMetaTagsForInfoToWriter(info, writer);

The MetaTags tag

The metaTags tag renders the meta tags with the information of a WebCrawlerInfo to a JSP. 

<%@ taglib prefix="contarini" uri="http://stefanbirkner.github.com/contarini/tags" %>
<contarini:metaTags info="${info}"/>
